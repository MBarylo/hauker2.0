# Mini Social Network (Hauker)

## 📌 Опис проєкту

Це full-stack веб-застосунок у форматі міні соціальної мережі, де користувачі можуть створювати пости, редагувати їх та видаляти.

## 🚀 Функціональність

- Реєстрація / логін користувача
- Створення постів
- Перегляд стрічки постів
- Редагування власних постів
- Видалення власних постів
- Пошук постів
- Валідація даних
- Обробка помилок

## 🛠 Технології

### Backend:

- Node.js
- NestJS
- TypeScript

### Frontend:

- React (Vite)
- TypeScript
- Axios
- Framer Motion

## ⚙️ Встановлення

### 1. Клонування репозиторію

```bash
git clone <your-repo-link>
cd project
```

### 2. Backend

```bash
cd backend
npm install
npm run start:dev
```

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🌐 API
